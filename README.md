# My-first-flask-GUI
Hello world GUI built using flask

## How to use;
1. Download the zip file  and extract my code
2. Instal flask using pip
  ''' pip install flask '''
3. Run the app.py
4. Then open the browser and type the link shown as,
  > http://###.#.#.#:5000 
          or
  > localhost:[the digits after the ":" sign ]
